# DC/V Experiment Reproducibility Notes

This export contains the final DC and V result table, the selected best checkpoints,
and source metadata for each exported run.

With this branch on another machine, you can reload the exported checkpoints and
reproduce the saved result tables when the matching benchmark inputs are available.

For a full from-scratch rerun, also keep the same raw/preprocessed datasets,
Python/PyTorch environment, training commands, split policy, random seeds, and GPU
runtime settings. The checkpoint package alone is not a complete raw-data archive.
