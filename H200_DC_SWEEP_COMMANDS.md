# H200 DC 測試指令手冊

這個分支是從 `codex/dc-training` 切出來的 DC 專用 H200 測試分支。

這裡新增的內容：

- `submit_nano5_stgat_dc.slurm` 支援顯式覆寫 `LR` 與 `MAX_TIME_STEPS`
- 這份 DC 專用測試手冊
- DC 專用 branch marker

這裡測的是：

- `DC` 流程
- `TRAIN_TASK=dc`
- `MONITOR_TASK=dc`

前提假設：

- Nano5 路徑：`/work/<your_account>/STDR-STGAT`
- 環境先建立一次：`bash create_nano5_env.sh`
- 建議同時最多只跑 `2` 個 job
- 這份手冊使用的 LR heuristic：
  - `LR = 1e-3 * sqrt(BATCH_SIZE / 32)`

## 先做一次環境準備

```bash
cd /work/<your_account>/STDR-STGAT
bash create_nano5_env.sh
```
